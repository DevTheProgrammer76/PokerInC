# PokerInC
This is a project for a coding class some friends and I made in C to create a playable poker game.

##Features
- Random Value assigning for cards
- Betting and folding functions
- Option menu with a rules and explanation option

##How to Play
1. Make sure you have a C programming engine installed
2. Download the files (PokerProject.c and pokerStruct.h)
3. Compile the code using gcc -ansi -Wall OR gcca
4. run the code using a.out
